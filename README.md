## Configuration Steps

Cloning the repository:
```
$ git clone https://github.com/williamtdepaula/flutter_todo_mobx
```

Open the project:
```
$ cd todo_list_mobx
```

Now run the app on your connected device (using terminal):
```
$ flutter run
```
